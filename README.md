# 课程服务需求登记页

这是一个适合发给客户填写的静态网页。

## 客户链接

直接把部署后的普通链接发给客户：

```text
https://你的域名.vercel.app
```

客户只会看到：

- 需求填写表单
- 提交后的需求摘要
- 复制需求摘要按钮

客户不会看到内部意向等级、推荐产品和推荐话术。

## 顾问内部链接

你自己查看内部判断时，在链接后面加：

```text
?owner=1
```

示例：

```text
https://你的域名.vercel.app?owner=1
```

这个页面会显示：

- 客户意向等级
- 推荐产品
- 推荐发送话术
- 是否建议继续沟通

## 重要说明

当前版本没有数据库。客户提交后，需要点击“复制需求摘要”，再通过微信发给你。

如果你希望客户提交后你自动收到，需要继续接入：

- 飞书表格
- 企业微信
- 邮箱
- 表单后台
