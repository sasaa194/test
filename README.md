# 客户需求收集网页

这是一个单页静态网页，适合直接部署到 Vercel、Netlify、GitHub Pages，然后把链接发给客户填写。

## 本地查看

直接双击 `index.html`，或者用浏览器打开即可。

## 推荐发布方式：Vercel

1. 把当前文件夹上传到 GitHub 仓库
2. 登录 Vercel
3. 选择导入这个仓库
4. 保持默认配置直接部署
5. 部署完成后会得到一个可分享链接

## 备用发布方式：Netlify

1. 登录 Netlify
2. 新建站点并上传整个文件夹
3. 部署完成后复制站点链接

## 发给客户的话术

你可以直接发：

`先填一下这个需求表，我看完后直接给你更准确的方案，能少来回沟通几轮。`
